# Fork Workflow

This repository follows a fork-based contribution workflow.

## Step 1: Fork

Click **Fork** on GitHub to create your own copy.

## Step 2: Clone

Clone your fork locally.


git clone https://github.com/YOUR_USERNAME/digital-logic-adders.git


## Step 3: Create Branch

Never work directly on the main branch.


git checkout -b feature/my-improvement


## Step 4: Commit

Write meaningful commit messages.

Example:


Add four-bit ripple adder testbench


## Step 5: Push


git push origin feature/my-improvement


## Step 6: Pull Request

Submit a pull request against the main repository.

Describe:

- what changed
- why the change exists
- how it was verified
